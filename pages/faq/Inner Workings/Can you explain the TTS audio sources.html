Understanding TTS Audio Sources

TTS Company uses two types of audio sources:


  • Global TTS Audio Source

  • Calling Assembly TTS Audio Source (mod-specific)

Each network object can have only one Global TTS Audio Source, but any number of Calling Assembly TTS Audio Sources, one per mod that adds one.

If a TTS audio source receives multiple speak requests before it finishes playing, only the most recent request will actually play, earlier ones are cancelled. This applies regardless of whether the requests came from the same mod or different mods.

This prevents TTS lines from overlapping and talking over each other on the same source, which is the desired behavior in most cases, for example, when a player character is speaking using TTS.

Sometimes, though, you do want TTS lines to be able to overlap. Since each Calling Assembly TTS Audio Source is separate from the Global one, requests made to the different audio sources can play simultaneously without cancelling each other.
