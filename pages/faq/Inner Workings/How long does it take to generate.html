Note

If the user's system is low on memory, some voice models will automatically unload in the background. Requesting an unloaded model forces the mod to swap models in and out of memory, which dramatically increases TTS generation time.

TTS generation speed depends on your hardware and the length of the TTS input text.

Testing showed that uncached TTS generation typically takes roughly 1 to 2 seconds.

Interestingly, generation time remains consistent regardless of hardware scaling, showing no noticeable difference between a 2-core and an 18-core CPU, but there might be more difference when using long TTS input texts.

If the same line is requested again using identical voice settings, TTS generation is instantaneous as the mod plays the cached audio file.
