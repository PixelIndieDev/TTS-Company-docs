Note

Memory usage for loaded TTS voices is managed automatically, you don't need to track it yourself.

Asynchronously loads a TTS voice model into memory so it's ready to generate speech.

Inputs

voiceModelName

The file name of the .onnx voice model to load

Returns

Success

true if the model loaded successfully, false otherwise

Error

"ok" if the operation succeeded, or an error message describing what went wrong

C# 
            
                public static async Task<(bool Success, string Error)> PreloadTTSVoiceModelInMemory(string voiceModelName)

Tip

Best practice example

C# 
                
                    // Preload the voice model in your mod's Awake() method
                    await TTSCompanyAPI.PreloadTTSVoiceModelInMemory("en_US-hfc_female-medium");

Note

These voice models are loaded by default and don't need to be preloaded manually:

C# 
                
                    "en_US-hfc_female-medium"
                    "en_US-hfc_male-medium"
