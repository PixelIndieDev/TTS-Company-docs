Note

Memory usage for loaded TTS voices is managed automatically, you don't need to track it yourself.

Note

Voices still needed by other mods won't be unloaded, this is handled automatically.

Asynchronously unloads a previously loaded TTS voice model from memory.

Inputs

voiceModelName

The file name of the .onnx voice model to unload

Returns

Success

true if the model unloaded successfully, false otherwise

Error

"ok" if the operation succeeded, or an error message describing what went wrong

C# 
            
                public static async Task<(bool Success, string Error)> UnloadTTSVoiceModelInMemory(string voiceModelName)
