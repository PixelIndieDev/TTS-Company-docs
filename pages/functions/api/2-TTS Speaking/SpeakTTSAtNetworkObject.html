Important

Load a TTS voice model first, and make sure the network object has a TTS audio source before speaking.

Note

Each item in a string[] is spoken one after another, in order.

Note

Text is automatically split by sentence before generation when inputting a single string.

Generates and plays TTS audio at a network object, if the TTS audio source is found on the network object.

Tip

Best practice example

C# 
                
                    // If a sentence repeats often but with one changing word, split it into separate array items.
                    // That way only the changing word needs to be generated. Everything else is reused from the cache.
                    // See the example below.

                    // Here we use a random enemy name as the changing word.
                    string[] enemyNames = new[]
                    {
                        "Barber",
                        "Bracken",
                        "Bunker Spider",
                        "Coil-Head",
                        "Hoarding Bug",
                        "Hygrodere",
                        "Jester",
                        "Nutcracker"
                    };

                    // The sentence being reused
                    string[] customStrings = new[]
                    {
                        "Warning, ",
                        "ENTITYNAME",
                        " detected near your position."
                    };

                    // Pick a random enemy name
                    randomIndex = UnityEngine.Random.Range(0, enemyNames.Length);
                    // Swap in the enemy name
                    customStrings[1] = enemyNames[randomIndex];

                    // Have the local player speak the line
                    PlayerControllerB localPlayer = StartOfRound.Instance.localPlayerController;
                    // Now speak
                    TTSCompanyAPI.SpeakTTSAtNetworkObject(localPlayer.gameObject, customStrings);
