Note

Text is automatically split by sentence before generation when inputting a single string.

Generates and caches TTS audio lines ahead of time for instant playback later. This does not play any audio.

Inputs

textsToSpeak Overload 1

textToSpeak Overload 2

A string containing the text lines to be converted to speech

voiceSettings optional

Returns

None

Tip

"Best" practice example

C# 
                
                    // Use this to pre-cache lines that need to play instantly, like reactions
                    // See the example below.

                    // In your mod's Awake() method
                    // Awake() isn't always the right place. Pick what fits your mod's usage.
                    await TTSCompanyAPI.PreloadTTSVoiceModelInMemory("en_US-hfc_female-medium");
                    
                    // Generate the TTS ahead of time
                    TTSCompanyAPI.PreGenerateTTS("Oh watch out!");
                    TTSCompanyAPI.PreGenerateTTS("Watch out, quicksand!");
