Preload TTS voice model in memory

Asynchronously loads a TTS voice model into system memory

Inputs

voiceModelName

The file name of the voice model to preload

Returns

Success

true if the model was successfully preloaded, false otherwise

Error

An error message if the operation failed, "ok" if successful

C# 
            
                public static async Task<(bool Success, string Error)> PreloadTTSVoiceModelInMemory(string voiceModelName)

Tip

Best practice example

C# 
                
                    // In your mod's Awake() method, preload the voice model
                    await TTSCompanyAPI.PreloadTTSVoiceModelInMemory("en_US-hfc_female-medium");

Note

The following voice models are loaded by default:

C# 
                
                    "en_US-hfc_female-medium"
                    "en_US-hfc_male-medium"
                    "en_US-ryan-medium"
                    "en_US-sam-medium"
