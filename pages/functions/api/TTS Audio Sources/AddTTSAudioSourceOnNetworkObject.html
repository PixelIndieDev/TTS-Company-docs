Important

Make sure you have loaded a TTS voice and given the network object an TTS audio source!

Adds a TTS audio source component to a network object, enabling it to generate and play TTS audio.

Inputs

networkObjectRefOfSpeaker Overload 1

A NetworkObjectReference pointing to the network object

objectRefOfSpeaker Overload 2

The local GameObject instance to add the TTS audio source to

useGlobalAudioSource optional

Determise whether to use the global TTS audio source or the calling assembly TTS audio source

audioSourceSettings optional

TTSAudioSourceSettings object containing configuration options for the TTS audio source (volume, pitch, spatial audio settings, etc.)

Returns

None
