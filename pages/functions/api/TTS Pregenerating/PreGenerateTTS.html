Note

The textToSpeak input will automatically be split by sentence under the hood.

Pre-generates and caches TTS audio lines for instant future use. This will not play the TTS.

Inputs

textsToSpeak Overload 1

An array of strings containing the text lines to be converted to speech

textToSpeak Overload 2

A string containing the text lines to be converted to speech

voiceSettings optional

PiperVoiceSettings object containing configuration options for the voice parameters (speaking rate, voice model, etc.)

Returns

None
C# (Overload 1) 
            
                public static void PreGenerateTTS(string[] textsToSpeak, PiperVoiceSettings voiceSettings = APIDefaultsConstants.PIPER_VOICE_SETTING_DEFAULT)
C# (Overload 2) 
            
                public static void PreGenerateTTS(string textToSpeak, PiperVoiceSettings voiceSettings = APIDefaultsConstants.PIPER_VOICE_SETTING_DEFAULT)

Tip

"Best" practice example

C# 
                
                    // This is best used for TTS that needs instant play, like for example reactions.
                    // See below for an example

                    // In your mod's Awake() method
                    // Depending on your use Awake() might not always be the best option.
                    await TTSCompanyAPI.PreloadTTSVoiceModelInMemory("en_US-hfc_female-medium");
                    
                    // Generate the TTS early
                    TTSCompanyAPI.PreGenerateTTS("Oh watch out!");
                    TTSCompanyAPI.PreGenerateTTS("Watch out, quicksand!");
