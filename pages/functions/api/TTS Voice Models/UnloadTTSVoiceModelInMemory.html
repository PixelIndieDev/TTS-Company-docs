Note

You don't need to worry about managing system memory usage with the amount of TTS voices loaded in. This library does that automatically.

Note

You don't need to worry about unloading TTS voices that are wanted by other mods. This library handles that automatically.

Asynchronously unloads a previously loaded TTS voice model from system memory.

Inputs

voiceModelName

The file name of the voice model to unload

Returns

Success

true if the model was successfully unloaded, false otherwise

Error

An error message if the operation failed, "ok" if successful
