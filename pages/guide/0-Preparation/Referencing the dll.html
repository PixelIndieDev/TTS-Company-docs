Reference the TTS Company DLL

Before you can call any TTS Company functions, add a reference to the TTS Company DLL in your mod project.

In Visual Studio:

  1. In the Solution Explorer, right-click References (or Dependencies, depending on your project type) under your project.
  2. Select Add Project Reference...
  3. In the dialog that opens, click Browse... in the bottom-left corner.
  4. Navigate to wherever TTSCompany.dll is located on your machine, commonly your game's BepInEx\plugins folder, select it, and click Add.
  5. Make sure the checkbox next to TTSCompany.dll is ticked, then click OK.

Then add the following to the top of any file that uses the API:

C# 
                
                    using TTSCompany;

Declare the dependency in your BepInPlugin

So BepInEx loads TTS Company before your mod, add a [BepInDependency] attribute above your plugin class, referencing the TTS Company plugin GUID:

C# 
                
                    [BepInPlugin(MyPlugin.PLUGIN_GUID, MyPlugin.PLUGIN_NAME, MyPlugin.PLUGIN_VERSION)]
                    [BepInDependency("PixelIndieDev_TTSCompany")]
                    public class MyPlugin : BaseUnityPlugin
                    {
                        // ...
                    }

Add the dependency to your manifest.json

If you're distributing your mod through Thunderstore, list TTS Company as a dependency in your manifest.json so it's installed automatically alongside your mod:

JSON 
                
                    {
                        "name": "YourModName",
                        "version_number": "1.0.0",
                        "website_url": "",
                        "description": "Your mod description",
                        "dependencies": [
                            "PixelIndieDev-TTSCompany-1.0.0"
                        ]
                    }

The dependency string follows the format TeamName-PackageName-Version. Make sure the version number matches (or is compatible with) the TTS Company release you built against.

That's it.

You've now made a network object speak TTS.
