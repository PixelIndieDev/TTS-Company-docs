Important

The example code omits null checks to keep things easy to follow. Add appropriate null checks in your own code.

Note

This guide assumes you're using one of the four default TTS voice models, which are already loaded and ready to use.

Add a TTS audio source to the network object you want to let speak.

In this example, we'll make the local player speak.

C# 
                
                    // Get the player
                    PlayerControllerB speakingPlayer = StartOfRound.Instance.localPlayerController;

                    // Add a TTS audio source
                    // We'll use the global audio source
                    TTSCompanyAPI.AddTTSAudioSourceOnNetworkObject(speakingPlayer.gameObject);

Now let's make the player speak the line: "This is a testing text-to-speech voice line."

C# 
                
                    // Get the player
                    PlayerControllerB speakingPlayer = StartOfRound.Instance.localPlayerController;

                    // Add a TTS audio source
                    // We'll use the global audio source
                    TTSCompanyAPI.AddTTSAudioSourceOnNetworkObject(speakingPlayer.gameObject);

                    // Now make the player speak
                    // This uses the player's global TTS audio source
                    TTSCompanyAPI.SpeakTTSAtNetworkObject(speakingPlayer.gameObject, "This is a testing text-to-speech voice line.");

That's it.

You've now made a network object speak TTS.
