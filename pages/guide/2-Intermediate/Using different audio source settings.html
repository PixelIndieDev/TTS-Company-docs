Using different audio source settings.

We'll continue from the example in Basic Usage.


First, let's create a new TTSAudioSourceSettings object.

C# 
                
                    PlayerControllerB speakingPlayer = StartOfRound.Instance.localPlayerController;

                    // You could pass a TTSAudioSourceSettings object directly into AddTTSAudioSourceOnNetworkObject(), but here we'll update the settings afterwards instead
                    TTSCompanyAPI.AddTTSAudioSourceOnNetworkObject(speakingPlayer.gameObject);

                    // Create a TTSAudioSourceSettings object
                    TTSAudioSourceSettings newTTSAudioSettings = new TTSAudioSourceSettings();

                    // Apply the new settings to the player's TTS audio source
                    TTSCompanyAPI.UpdateTTSAudioSourceSettingsOnNetworkObject(speakingPlayer.gameObject, newTTSAudioSettings);

                    TTSCompanyAPI.SpeakTTSAtNetworkObject(speakingPlayer.gameObject, "This is a testing text-to-speech voice line.");

Now let's make some changes

C# 
                
                    PlayerControllerB speakingPlayer = StartOfRound.Instance.localPlayerController;

                    // You could pass a TTSAudioSourceSettings object directly into AddTTSAudioSourceOnNetworkObject(), but here we'll update the settings afterwards instead
                    TTSCompanyAPI.AddTTSAudioSourceOnNetworkObject(speakingPlayer.gameObject);

                    // Create a TTSAudioSourceSettings object
                    TTSAudioSourceSettings newTTSAudioSettings = new TTSAudioSourceSettings();
                    // Lower the volume
                    newTTSAudioSettings.Volume = 0.5f;
                    // Make the audio play in 2D instead of 3D
                    newTTSAudioSettings.SpatialBlend = 0.0f;
                    // Add some Doppler effect
                    newTTSAudioSettings.DopplerLevel = 0.75f;

                    // Apply the new settings to the player's TTS audio source
                    TTSCompanyAPI.UpdateTTSAudioSourceSettingsOnNetworkObject(speakingPlayer.gameObject, newTTSAudioSettings);

                    TTSCompanyAPI.SpeakTTSAtNetworkObject(speakingPlayer.gameObject, "This is a testing text-to-speech voice line.");

That's it.

The TTS will now play using your new audio settings.
