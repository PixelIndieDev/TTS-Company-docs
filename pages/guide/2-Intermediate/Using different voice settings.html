Using different voice settings.

We'll continue from the example in Basic Usage.


First, let's create a new PiperVoiceSettings object.

C# 
                
                    PlayerControllerB speakingPlayer = StartOfRound.Instance.localPlayerController;

                    TTSCompanyAPI.AddTTSAudioSourceOnNetworkObject(speakingPlayer.gameObject);

                    // Create a PiperVoiceSettings object
                    PiperVoiceSettings newVoiceSettings = new PiperVoiceSettings();

                    // Don't forget to pass the new voice settings into the speak function
                    TTSCompanyAPI.SpeakTTSAtNetworkObject(speakingPlayer.gameObject, "This is a testing text-to-speech voice line.", voiceSettings: newVoiceSettings);

Now let's make some changes

C# 
                
                    PlayerControllerB speakingPlayer = StartOfRound.Instance.localPlayerController;

                    TTSCompanyAPI.AddTTSAudioSourceOnNetworkObject(speakingPlayer.gameObject);

                    // Create a PiperVoiceSettings object
                    PiperVoiceSettings newVoiceSettings = new PiperVoiceSettings();
                    // Switch to a different voice model
                    // We'll use one of the default loaded models: 'en_US-ryan-medium'
                    newVoiceSettings.ModelName = "en_US-ryan-medium";
                    // Make the speech rate slightly slower than normal
                    newVoiceSettings.SpeechRate = 1.2f;
                    // Increase pitch and intonation variance for more expressiveness
                    newVoiceSettings.NoiseScale = 0.8f;
                    // Reduce phoneme timing variance for a slightly more rigid cadence
                    newVoiceSettings.NoiseScaleW = 0.5f;
                    // Add a 0.5 second pause before each new sentence
                    newVoiceSettings.SentenceSilence = 0.5f;
                    // Remove pauses after commas and other mid-sentence punctuation
                    newVoiceSettings.PunctuationSilence = 0.0f;

                    // Don't forget to pass the new voice settings into the speak function
                    TTSCompanyAPI.SpeakTTSAtNetworkObject(speakingPlayer.gameObject, "This is a testing text-to-speech voice line.", voiceSettings: newVoiceSettings);

That's it.

The TTS will now speak using your new voice settings.
