Note

The float values are rounded to 2 decimals.

Config options:

TTS generation priority

Adjusts the CPU priority for generating TTS. Higher priority may increase the TTS generation speed at the cost of performance.

Default: TTSGenPriority.Normal

TTS timeout scaling

Controls how long the mod waits for a voice line to generate. Increase this if your TTS audio keeps getting cut off, or decrease it if you want the mod to give up faster when experiencing delays. Controlled by the host.

Default: TimeoutBufferScaling.Normal

Clear TTS cache on exit

When enabled, automatically deletes saved TTS cache when you close Lethal Company. Disabling this saves disk space but requires files to be regenerate next session.

Default: false

Clear TTS cache now

Check this checkbox to delete all saved local TTS cache. Best done in the main menu.

Default: false
