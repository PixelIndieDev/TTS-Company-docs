Note

The float values are rounded to 2 decimals.

Inputs

ModelName

The file name of the .onnx voice model to use for speech generation

Default: ExampleConstants.VOICE_MODEL_NAME

SpeechRate

Multiplier for how fast the voice speaks (values below 1.0 speed up speech, values above 1.0 slow it down, e.g. 0.5 = twice as fast, 2.0 = half speed)

Default: 1.0f

Min: 0.05f

Max: 5.0f

NoiseScale

Controls pitch and intonation variation (higher values make the voice sound more expressive and varied, lower values make it more flat and monotone)

Default: 0.67f

Min: 0.0f

Max: 1.0f

NoiseScaleW

Controls variation in phoneme timing and pacing (higher values add natural rhythmic variety, lower values produce a stiffer more robotic cadence)

Default: 0.8f

Min: 0.0f

Max: 1.0f

SentenceSilence

The pause, in seconds, inserted after sentence-ending punctuation ("!", ".", "?", etc)

Default: 0.2f

Min: 0.0f

Max: 5.0f

PunctuationSilence

The pause, in seconds, inserted after mid-sentence punctuation (",", ":", ";", etc)

Default: 0.08f

Min: 0.0f

Max: 2.5f
