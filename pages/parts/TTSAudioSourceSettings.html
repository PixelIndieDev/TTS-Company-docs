Note

The float values are rounded to 2 decimals.

Wraps Unity's AudioSource settings so you can control how the generated TTS audio is played back.

Inputs

BypassEffects

If true, skips any audio filter effects (e.g. reverb, distortion) applied to this audio source's GameObject

Default: false

BypassListenerEffects

If true, skips any effects applied to the Audio Listener

Default: false

BypassReverbZones

If true, ignores any reverb zones the audio would otherwise pass through

Default: false

Priority

Playback priority relative to other sounds (Lower values are higher priority, 0 is highest priority, 256 is lowest)

Default: 128

Min: 0

Max: 256

Volume

The playback volume

Default: 1.0f

Min: 0.0f

Max: 1.0f

SpatialBlend

How much the sound is treated as 3D (positional) versus 2D (0.0 is fully 2D, 1.0 is fully 3D/spatial)

Default: 1.0f

Min: 0.0f

Max: 1.0f

ReverbZoneMix

How much of the audio signal is routed to reverb zones

Default: 1.0f

Min: 0.0f

Max: 1.0f

DopplerLevel

How strong the Doppler effect is when the source or listener is moving

Default: 1.0f

Min: 0.0f

Max: 1.0f

MinDistance

The distance within which the sound stays at full volume without attenuating (3D sounds only)

Default: 1.0f

Min: 0.0f

Max: 128.0f

MaxDistance

The distance beyond which the sound stops attenuating further (3D sounds only)

Default: 40.0f

Min: 1.0f

Max: 128.0f

RolloffMode

How the volume fades out with distance between MinDistance and MaxDistance (e.g. Linear, Logarithmic, or Custom)

Default: AudioRolloffMode.Linear

OutputAudioMixerGroup

The Audio Mixer Group this source routes its output through, if any

Default: null

CustomCurve

A custom volume-over-distance curve, used when RolloffMode is set to Custom

Default: null
